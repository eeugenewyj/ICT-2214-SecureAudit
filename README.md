# ICT-2214-SecureAudit


### **To use HTML Crawler**:
1. Ensure that website is up and running
2. Go into folder with scanner_gui and run "python3 scanner_gui.py"
3. Input full link into "Target Input" (for example; https://local.test.com) and then click analyze
4. Output should be shown in the 3 boxes
5. Click "Export Report" to export into a readable json file, "Clear" to clear output and reanalyze
